## 🤔 What Is This?

This is **not** a list of AI experiments or research demos.

Every tool in this repo has been built to solve a **real problem**, in a **real industry**, for **real businesses**. These are the same tools built and deployed at [RVS Media](https://rvsmedia.co.uk) and [Velocity AI](https://theaishift.co) for clients across ecommerce, marketing, recruitment, education, and beyond.

We open source them because the best way to prove AI works is to **let people use it themselves.**

## 🚀 Getting Started

**Step 1 — Clone the repo**
```bash
git clone https://github.com/rajeevnar/the-ai-shift.git
```

**Step 2 — Navigate to your chosen tool**
```bash
cd the-ai-shift/marketing/linkedin-newsletter-generator
```

**Step 3 — Follow the tool's README**

Each tool has its own `README.md` with setup instructions. Most run in under 5 minutes.

**Step 4 — Get your API key**

Most tools are powered by Claude AI. Get your free API key at [console.anthropic.com](https://console.anthropic.com)
